# AVM Constructions

**We Build Today... You Live Tomorrow.**  
A modern, responsive, and beautifully designed web application for AVM Constructions, delivering quality construction and elegant interiors across Vriddhachalam and Andimadam.

## 🚀 Overview

This website serves as the digital storefront for AVM Constructions. It showcases the company's portfolio, details their service offerings, transparently displays construction pricing packages, and provides seamless contact channels for prospective clients.

Built with performance, SEO, and stunning user experience (UX) in mind, the application features dynamic hover effects, mobile-first responsive layouts (including horizontal swipe carousels), and native mobile OS integrations like direct Google Maps linking.

## ✨ Key Features

- **Modern & Premium Design:** Sleek, glassmorphism-inspired UI with carefully curated color palettes and dynamic micro-animations to engage visitors.
- **Responsive Portfolio Carousel:** A swipeable, fully responsive "Our Works" gallery showcasing past construction and interior projects.
- **Interactive Pricing Packages:** Clear, card-based layouts detailing structural and finishing costs per square foot, complete with interactive UI feedback.
- **Native Mobile Integrations:** Seamless phone, email, and Google Maps links that open directly in native mobile applications.
- **Performance Optimized:** Built with Next.js App Router and Tailwind CSS for lightning-fast loading speeds and optimal SEO.

## 🛠️ Technology Stack

- **Framework:** [Next.js (App Router)](https://nextjs.org/)
- **Library:** [React](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Icons & Assets:** Optimized SVGs and RealFaviconGenerator

## 💻 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine. 

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/madhumica-dev/avmconstructions.git
   cd avmconstructions
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **View the site**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🏗️ Project Structure

- `/src/app` - Contains the main Next.js App Router layouts, pages, and global stylesheets.
- `/src/components` - Reusable UI components grouped by feature (e.g., `home`, `portfolio`, `layout`, `ui`).
- `/src/data` - Static data structures (like the `projects` list for the portfolio).
- `/public` - Static assets including images, favicons, and Web App Manifests.
